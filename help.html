Tutorial

Administrator Page

  1. Press the 'Start' button to begin polling
    • The timer will start counting up
    • The Respondent page will now accept answers
  2. Press the 'Stop' button when you want to stop the polling
    • The Respondent page will no longer accept answers
  3. Press the 'Show Results' button to display the responses to the poll
    • The responses will be displayed in the form of a bar graph on the right side of the page
    • The height of each of the bars will be proportionate to the number of responses given for each of the options
  4. Press the 'Start' button again to start a new poll
    • The timer will reset and start counting up from 00:00:00 again
    • The Respondent page will once again start accepting answers

Respondent Page

  1. Press/Tap on one of the choices
    • The selected choice will be grayed out
    • The ‘Submit’ button will no longer be grayed out and will be active
  2. Press the ‘Submit’ button to send your response
    • The ‘Submit’ button will become grayed out and inactive
    • No more answers can be submitted until a new poll is started
